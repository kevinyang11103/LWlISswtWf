# 停车场管理系统 java86

## 项目概述

停车场管理系统是一款基于Java和Vue的前后端分离的web网站应用。该系统致力于实现高效、便捷的停车场管理功能。

## 技术栈

### 系统架构

- B/S架构
- MVC模式

### 运行环境

- 系统环境：Windows/Mac

### 开发环境

- 开发工具：IDEA
- JDK版本：1.8
- 构建工具：Maven
- 数据库：Mysql

### 技术框架

- 后端：Java、SpringBoot、Mybatis
- 前端：Vue

## 功能模块

### 用户模块

- 用户登录
- 用户注册

### 车主模块

- 个人中心
- 预定停车位
- 缴费信息

### 管理员模块

- 用户信息管理
- 车位信息管理
- 车位费用管理
- 停泊车辆管理
- 车辆进出管理
- 登录日志查询

## 系统角色

- 用户
- 车主
- 管理员
